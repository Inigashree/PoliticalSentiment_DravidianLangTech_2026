# TamilEcho @ DravidianLangTech 2026
### Hybrid XLM-RoBERTa with Sarcasm-Aware Feature Fusion for Political Multiclass Sentiment Analysis in Tamil X (Twitter) Comments

This repository contains the implementation of our system submitted to the **DravidianLangTech 2026 Shared Task on Political Multiclass Sentiment Analysis in Tamil X (Twitter) Comments**.

The proposed model combines **XLM-RoBERTa contextual embeddings**, **TF-IDF lexical features**, and **emoji-based sarcasm indicators** to improve sentiment classification in Tamil political tweets.

## 📊 Task Description

The goal of the shared task is to classify Tamil political tweets into **7 sentiment categories**:

- Positive
- Negative
- Sarcastic
- Opinionated
- Substantiated
- Neutral
- None of the Above
